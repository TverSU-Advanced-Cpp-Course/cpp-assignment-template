Этот репозиторий содержит шаблон для заданий общего назначения. Подробное описание содержимого появится тут позднее (может быть).

## Информация для составителей заданий

### Скрытые тесты

Этот шаблон включает настройки Github Workflow для запуска скрытых тестов при помощи Github Classroom Assignment Secrets.

Чтобы они заработали нужно:
1. Создать задание в Github Classroom
2. Добавить Assignment Secret:
    - Name: `INFRA_TESTS_TOKEN`
    - Value: PAT с доступом на чтение к `course-infra-tests`

Студенты автоматически получат этот секрет в их репозиториях и скрытые тесты будут запускаться на CI/CD после каждого пуша в репозиторий.
